# InternTrack Pro

A professional Internship Application Tracking System built using React, Spring Boot, MongoDB, and JWT Authentication.

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes

### Dashboard
- Total Applications
- Pending Applications
- Interview Applications
- Accepted Applications
- Rejected Applications
- Pie Chart Analytics
- Bar Chart Analytics
- Recent Applications
- Upcoming Interviews

### Company Management
- Add Company
- Update Company
- Delete Company
- Company Directory

### Application Management
- Add Internship Applications
- Update Applications
- Delete Applications
- Application Status Tracking
- Interview Scheduling
- Notes Management

## Tech Stack

### Frontend
- React.js
- Vite
- Axios
- React Router DOM
- Recharts
- CSS3

### Backend
- Spring Boot
- Spring Security
- JWT Authentication
- Lombok
- Maven

### Database
- MongoDB Atlas
